Project Detail

Strip Loaded Waveguide on LNOI

This project studies strip-loaded waveguide configurations built on the lithium niobate on insulator platform. The work is centered on achieving effective optical confinement while preserving a fabrication flow that is practical for device integration and scalable photonic design.

The broader objective is to evaluate how strip loading can support low-loss guiding, compact routing, and compatibility with future active or hybrid photonic components on LNOI.

Project Scope

The project includes structural design, simulation-informed geometry selection, and evaluation of the optical behavior expected from the strip-loaded architecture. It can also accommodate future comparison with conventional etched waveguides on the same material platform.

Current Focus

  • Assessing mode confinement and effective index behavior.
  • Balancing optical performance with fabrication simplicity.
  • Preparing the design for downstream device integration.

Expected Outcome

The expected outcome is a practical waveguide design route that supports robust photonic device development on LNOI, especially in applications where fabrication tolerance and integration flexibility are important.
